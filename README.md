# \# NyDIA



**Descripción**



NyDIA es una aplicación web orientada a la recomendación de pautas nutricionales y deportivas. Mediante los datos biológicos de cada usuario y la consulta que se realice al asistente virtual, se realizan llamadas API a un modelo de inteligencia artificial generativa que proporcionará las indicaciones correspondientes al usuario para lograr sus objetivos. 




**Guía de uso**

(url)

El usuario debe registrarse para poder usar la aplicación. Una vez registrado y rellenado el formulario con los datos, se debe iniciar sesión para comenzar un chat con el asistente. Se inicia sesión desde la página principal.
Cuando el usuario acceda a la aplicación correctamente, se muestra un chat con el cual se interactuará con la IA para recibir las recomendaciones pertinentes.
